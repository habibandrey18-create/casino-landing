    ЗАГРУЗКА НА GITHUB БЕЗ УСТАНОВКИ GIT (ЧЕРЕЗ БРАУЗЕР)
═══════════════════════════════════════════════════════════════════════════════

Если не хотите устанавливать Git, можно загрузить файлы через браузер!

═══════════════════════════════════════════════════════════════════════════════
ШАГ 1: СОЗДАЙТЕ РЕПОЗИТОРИЙ НА GITHUB
═══════════════════════════════════════════════════════════════════════════════

1. Зайдите на https://github.com
2. Войдите в аккаунт (habibandrey18)
3. Нажмите "+" (в правом верхнем углу) → "New repository"
4. Заполните:
   - Repository name: casino-landing
   - Description: (можно оставить пустым)
   - Public (выберите Public)
   - НЕ ставьте галочки:
     ❌ Add a README file
     ❌ Add .gitignore
     ❌ Choose a license
5. Нажмите "Create repository"

═══════════════════════════════════════════════════════════════════════════════
ШАГ 2: ЗАГРУЗИТЕ ФАЙЛЫ ЧЕРЕЗ БРАУЗЕР
═══════════════════════════════════════════════════════════════════════════════

1. На странице созданного репозитория вы увидите инструкции
2. Найдите кнопку "uploading an existing file" (загрузка существующего файла)
   Или просто перейдите по ссылке:
   https://github.com/habibandrey18/casino-landing/upload

3. Откройте папку проекта:
   C:\Users\NewAdmin\Desktop\сайт

4. Выберите ВСЕ файлы и папки, КРОМЕ:
   ❌ node_modules (не загружайте эту папку!)
   ❌ .env (не загружайте этот файл!)
   ❌ analytics.db (не загружайте базу данных!)
   ❌ *.log (файлы логов)
   ❌ Thumbs.db
   ❌ .DS_Store

5. Перетащите выбранные файлы в браузер (в область загрузки)

6. Внизу страницы:
   - Commit message: Initial commit for Render
   - Нажмите "Commit changes"

7. Дождитесь загрузки (1-2 минуты)

═══════════════════════════════════════════════════════════════════════════════
ШАГ 3: ПОДКЛЮЧИТЕ К RENDER
═══════════════════════════════════════════════════════════════════════════════

Теперь можно подключить репозиторий к Render:

1. Зайдите на https://render.com
2. Войдите через GitHub
3. New + → Web Service
4. Выберите репозиторий: habibandrey18/casino-landing
5. Настройки (см. RENDER_ИНСТРУКЦИЯ.txt)
6. Create Web Service

═══════════════════════════════════════════════════════════════════════════════
КАКИЕ ФАЙЛЫ ЗАГРУЖАТЬ
═══════════════════════════════════════════════════════════════════════════════

✅ ЗАГРУЖАЙТЕ:
   - index.html
   - admin.html
   - server.js
   - package.json
   - package-lock.json
   - casinos-data.js
   - tabs-manager.js
   - tracking.js
   - .gitignore
   - render.yaml
   - .nvmrc
   - README.md
   - Все .txt файлы с инструкциями
   - Папку images/ (если есть)

❌ НЕ ЗАГРУЖАЙТЕ:
   - node_modules/ (слишком большая папка)
   - .env (содержит пароли!)
   - analytics.db (база данных)
   - *.log (логи)
   - Thumbs.db
   - .DS_Store

═══════════════════════════════════════════════════════════════════════════════
ОБНОВЛЕНИЕ ФАЙЛОВ (ЕСЛИ НУЖНО ИЗМЕНИТЬ)
═══════════════════════════════════════════════════════════════════════════════

1. На GitHub откройте файл, который нужно изменить
2. Нажмите иконку карандаша (Edit this file)
3. Внесите изменения
4. Внизу страницы:
   - Commit message: опишите изменения
   - Нажмите "Commit changes"
5. Render автоматически обновит сайт
