  КАК ЗАГРУЗИТЬ НОВЫЕ ОБНОВЛЕНИЯ НА GITHUB И RENDER
═══════════════════════════════════════════════════════════════

📋 ПОШАГОВАЯ ИНСТРУКЦИЯ:

1. ОТКРОЙТЕ POWERSHELL В ПРАВИЛЬНОЙ ПАПКЕ
   ─────────────────────────────────────────────
   • Откройте папку: C:\Users\NewAdmin\Desktop\сайт
   • Нажмите правой кнопкой мыши в папке
   • Выберите "Открыть в терминале" или "Open in Terminal"
   • ИЛИ откройте PowerShell и выполните:
     cd "C:\Users\NewAdmin\Desktop\сайт"


2. ПРОВЕРЬТЕ ИЗМЕНЕНИЯ
   ─────────────────────────────────────────────
   Выполните команду:
   
   git status
   
   Эта команда покажет, какие файлы были изменены.


3. ДОБАВЬТЕ ВСЕ ИЗМЕНЕНИЯ
   ─────────────────────────────────────────────
   Выполните команду:
   
   git add .
   
   Эта команда добавит ВСЕ измененные файлы для отправки.


4. СОЗДАЙТЕ КОММИТ (СОХРАНЕНИЕ ИЗМЕНЕНИЙ)
   ─────────────────────────────────────────────
   Выполните команду:
   
   git commit -m "Улучшена админ-панель: добавлено автосохранение статистики и история"
   
   ИЛИ используйте свой текст описания изменений.


5. ОТПРАВЬТЕ НА GITHUB
   ─────────────────────────────────────────────
   Выполните команду:
   
   git push origin main
   
   Эта команда отправит все изменения на GitHub.


6. ПРОВЕРЬТЕ НА GITHUB
   ─────────────────────────────────────────────
   • Откройте: https://github.com/habibandrey18-create/casino-landing
   • Убедитесь, что ваши изменения появились


7. RENDER ОБНОВИТСЯ АВТОМАТИЧЕСКИ
   ─────────────────────────────────────────────
   • Render автоматически обнаружит изменения на GitHub
   • Начнется автоматическая сборка и развертывание
   • Обычно это занимает 2-5 минут
   • Проверьте статус на: https://dashboard.render.com


═══════════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════════

Если вы уверены, что хотите отправить ВСЕ изменения сразу:

git add .
git commit -m "Обновления админ-панели"
git push origin main

  ЧТО ДЕЛАТЬ, ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
═══════════════════════════════════════════════════════════════

❌ Ошибка "git is not recognized"
   → Git не установлен или не добавлен в PATH
   → Установите Git: https://git-scm.com/download/win

❌ Ошибка "fatal: not a git repository"
   → Вы не в правильной папке
   → Убедитесь, что находитесь в папке "сайт"

❌ Ошибка "Authentication failed"
   → Нужно авторизоваться в GitHub
   → Используйте Personal Access Token вместо пароля

❌ Ошибка "Everything up-to-date"
   → Это нормально! Все изменения уже отправлены
   → Проверьте на GitHub, что файлы обновлены

  ПРОВЕРКА УСПЕШНОЙ ЗАГРУЗКИ

После выполнения команд проверьте:

✅ git status должен показать "Your branch is up to date"
✅ На GitHub должны быть видны новые коммиты
✅ На Render должна начаться автоматическая сборка
✅ Через 2-5 минут сайт обновится с новыми изменениями
