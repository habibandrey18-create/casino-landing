═══════════════════════════════════════════════════════════════════════════════
ШАГ 1: РЕГИСТРАЦИЯ НА RENDER
═══════════════════════════════════════════════════════════════════════════════

1. Откройте браузер
2. Перейдите: https://render.com
3. Нажмите "Get Started for Free" (Начать бесплатно)
4. Нажмите "Continue with GitHub" (Войти через GitHub)
5. Войдите в аккаунт GitHub (habibandrey18-create)
6. Нажмите "Authorize Render" (Разрешить Render)
   - Это даст Render доступ к вашим репозиториям

═══════════════════════════════════════════════════════════════════════════════
ШАГ 2: СОЗДАНИЕ WEB SERVICE
═══════════════════════════════════════════════════════════════════════════════

1. На главной странице Render нажмите "New +" (вверху справа)
2. Выберите "Web Service" (Веб-сервис)

3. В разделе "Connect a repository":
   - Найдите репозиторий: habibandrey18-create/casino-landing
   - Если не видите - нажмите "Configure account" и разрешите доступ
   - Нажмите "Connect" рядом с репозиторием

═══════════════════════════════════════════════════════════════════════════════

⚠️ ВАЖНО: Скопируйте команды точно как написано!

4. НЕ нажимайте "Create Web Service" пока!
   Сначала нужно добавить переменные окружения (см. ШАГ 4)

ШАГ 4: ДОБАВЛЕНИЕ ПЕРЕМЕННЫХ ОКРУЖЕНИЯ
═══════════════════════════════════════════════════════════════════════════════

⚠️ БЕЗ ЭТИХ ПЕРЕМЕННЫХ САЙТ НЕ БУДЕТ РАБОТАТЬ!

В форме создания сервиса найдите раздел "Environment Variables"
(Переменные окружения)

Нажмите "Add Environment Variable" и добавьте по одной:

┌─────────────────────┬──────────────────────────────────────────────┐
│ Key                 │ Value                                        │
├─────────────────────┼──────────────────────────────────────────────┤
│ ADMIN_PASSWORD      │ Andrepa12345                                 │
│                     │ (ваш пароль для админ-панели)                 │
├─────────────────────┼──────────────────────────────────────────────┤
│ JWT_SECRET          │ casino-secret-key-2024-ultra-secure-random   │
│                     │ (любая случайная строка, минимум 20 символов)│
├─────────────────────┼──────────────────────────────────────────────┤
│ PORT                │ 10000                                        │
│                     │ (порт для Render)                            │
├─────────────────────┼──────────────────────────────────────────────┤
│ NODE_ENV            │ production                                   │
│                     │ (режим работы)                               │
└─────────────────────┴──────────────────────────────────────────────┘

Для каждой переменной:
1. Key: введите название (например, ADMIN_PASSWORD)
2. Value: введите значение (например, Andrepa12345)
3. Нажмите "Add"

═══════════════════════════════════════════════════════════════════════════════
ШАГ 5: СОЗДАНИЕ СЕРВИСА
═══════════════════════════════════════════════════════════════════════════════

1. Проверьте, что все 4 переменные добавлены
2. Прокрутите вниз
3. Нажмите "Create Web Service" (Создать веб-сервис)

4. Render начнет развертывание:
   - Скачает код с GitHub
   - Установит зависимости (npm install)
   - Запустит сервер (npm start)
   - Это займет 2-5 минут

5. Следите за процессом в разделе "Logs" (Логи)
   - Должны увидеть: "Build successful"
   - Затем: "Your service is live"

ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
═══════════════════════════════════════════════════════════════════════════════

❌ ПРОБЛЕМА: "Build failed"
   РЕШЕНИЕ:
   1. Откройте вкладку "Logs" в Render
   2. Прочитайте ошибку
   3. Проверьте, что Build Command: npm install
   4. Проверьте, что Start Command: npm start

❌ ПРОБЛЕМА: Сайт не открывается
   РЕШЕНИЕ:
   1. Подождите 1-2 минуты (первый запуск медленный)
   2. Проверьте логи в Render
   3. Убедитесь, что все 4 переменные окружения добавлены
   4. Проверьте, что переменные без лишних пробелов

❌ ПРОБЛЕМА: Пароль админки не работает
   РЕШЕНИЕ:
   1. В Render откройте ваш сервис
   2. Перейдите в "Environment"
   3. Проверьте переменную ADMIN_PASSWORD
   4. Убедитесь, что значение: Andrepa12345 (без пробелов)
   5. Сохраните изменения
   6. Render автоматически перезапустит сервис

❌ ПРОБЛЕМА: Сайт "засыпает"
   РЕШЕНИЕ:
   - Это нормально для бесплатного плана Render
   - Сервер "засыпает" после 15 минут бездействия
   - Первый запрос после простоя будет медленным (10-30 сек)
   - Это бесплатно, но если нужно постоянно - используйте VPS
